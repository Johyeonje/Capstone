우리팀의 아이디어에서는 한번에 여러명의 사진을 찍고 하나의 사진 안에 기존에 등록된 몇명의 인원이 포함되어있나 확인하는 절차인데, 그러려면 Face Detection이 필수적이다. 하지만 해당 기술은 우리가 새로 만들어 내기에 난이도가 너무 높고, 이미 해당 기술에 대한 오픈소스가 존재하기 때문에 우리 아이디어에 적합한 수준의 Face Detection 오픈소스를 탐색하고 활용하기로 하였다.

본 코드는 Open CV에서 공개한 Face Detection 오픈소스로 탐색 방식은 사람의 눈의 위치를 탐색하여 눈을 기준으로 기존 사진안의 얼굴의 좌측 상단과 우측 하단의 픽셀 위치를 반환하여 사각형의 도형을 얼굴이라고 인식된 부분에 그려준다.

haarcascade_frontalface_default : OpenCV에서 FaceDetection 에 사용되는 변수를 저장해놓은 xml 형식파일로 Detection 함수에 호출된다.
